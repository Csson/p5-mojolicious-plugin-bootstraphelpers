The class is set to .panel-default, by default:

%= panel
<div class="panel panel-default">
    <div class="panel-body">
    </div>
</div>

If you want a panel without title, set the title to undef:

%= panel undef ,=> begin
    <p>A short text.</p>
%  end
<div class="panel panel-default">
    <div class="panel-body">
        <p>A short text.</p>
    </div>
</div>

<%= panel 'The Header', -header => [
                        buttongroup => [buttons => [
                            ['Click me', ['#'], data => { holder => 'claw'}],
                            ['Tap-tap', ['#'], caret, items => [
                                ['Me too', ['#']]
                            ]],
                        ]]] => begin %>
    <p>A short text.</p>
%  end
